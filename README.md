# 江苏省招生计划变化知识库｜V1.1.44 专业详情点击增强版

本版基于 V1.1.43 继续迭代，只增强“具体专业详情”交互，不改行级权威数据口径。

## 本版改动

- 专业主表中，专业名称改成可点击入口。
- 专业主表中新增“详情”按钮，点击弹出专业详情。
- 志愿表已选专业列表中新增“详情”按钮。
- 志愿表专业池中每个专业新增“详情”按钮。
- 专业详情弹层补充“专业风险与梯队提示”，同时继续显示：
  - 院校基础信息
  - 专业组基础信息
  - 专业基础信息
  - 硕博点
  - 学科评估
  - 软科评级 / 排名
  - 招生录取数据
  - 批注信息

## 保留内容

- 行级权威数据口径
- 预估分重算
- 专业组变迁
- 保研双非筛选
- 专业优先标红
- 组内加权排序
- 纯圆形回顶悬浮球
- 刺客专业 v0.2 + 严格梯队补充规则
