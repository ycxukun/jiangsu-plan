# V1.1.35 专业组变迁与保研筛选版

本版本基于 V1.1.34 继续迭代，保留行级权威数据口径：分数、计划、专业基础信息、院校基础信息仍以专家版 Excel 每一行数据为准。

## 本版新增

1. 专业组历史变迁层
   - 新增 `data-group-changes.js`。
   - 变迁层只解释 2025→2026 专业组结构变化，不覆盖行级权威数据。
   - 专业组卡片与专业组明细里的“变迁”按钮可查看前身组、专业新增、减少、拆入、拆出、置信度和核对建议。

2. 院校层次筛选增加保研
   - 院校层次筛选面板新增“保研”分组。
   - 支持：保研资格、保研率≥10%、≥15%、≥20%、≥25%、≥30%。

## 审计文件

- `group_change_mapping_summary.csv`：变迁映射统计。
- `group_change_mapping_audit.csv`：当前专业组与变迁分析键的映射明细。
- 其余 `identity_key_*`、`row_authoritative_*` 文件继续保留用于行级数据核验。

## 数据口径

专业组变迁只作为解释层使用。任何分数、位次、计划、学费、硕博点、保研率等字段仍以行级权威数据为准。
