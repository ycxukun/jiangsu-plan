# 江苏省招生计划变化知识库｜V15 管理员在线保存版

## 本版修改

1. 普通访问仍然是只读模式。
2. 管理员入口：URL 加 `?admin=1` 或 `#admin`。
3. 管理员模式下可以连接 GitHub。
4. 右键学校 / 专业组 / 专业写备注后，可自动同步到仓库根目录的 `annotations.json`。
5. 不再必须手动导出、复制、上传 annotations.json。
6. GitHub Token 只保存在浏览器 sessionStorage，不写入网页文件。
7. 三年均分/位次已合并为一列。
8. 固定备注面板、详情页备注保留。
