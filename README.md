# V1.1.34 筛选交互优化版

本版沿用 V1.1.32/33 的行级权威数据口径：网页骨架保留，所有填充字段仍以专家版 Excel 的每一行为准。

## 本版主要调整

1. 院校基础信息入口轻量化
   - 去掉重黑框样式。
   - 改为浅绿色文字 + 细下划线入口：院校基础信息 ▾。

2. 多选筛选改为分组抽屉
   - 地区按华东、华北、华中、华南、西南、西北、东北分组。
   - 院校层次按国家层次、常规层次等分组。
   - 选科要求按常用/组合要求分组。
   - 专业大类按学科门类分组。

3. 筛选交互优化
   - 支持面板内搜索。
   - 支持分组全选、分组清空。
   - 顶部显示已选项，点击已选项可直接移除。
   - 底部固定“清空 / 取消 / 应用筛选”。
   - 勾选过程中不立即刷新主页面，点击“应用筛选”后再统一刷新。
