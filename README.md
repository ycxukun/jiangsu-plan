# GitHub Pages 静态部署版｜V1.1.36

本版在 V1.1.35 基础上新增：

1. 专业优先/专业清类录取规则红色提示：
   - 根据招生规则字段自动识别“专业优先、专业志愿清、专业清、志愿优先”等；
   - 对中国地质大学等已知需重点核对的学校增加人工红色提示；
   - 在院校索引、院校标题、专业组标签处同步标红。

2. 回到顶部改为悬浮球：
   - 右下角圆形悬浮按钮；
   - 上移位置，避免和其他右下角元素冲突。

3. 同一学校内专业组排序：
   - 不再按专业组投档线排序；
   - 改为按该专业组内所有专业的 25 年专业最低分 × 25 年计划人数加权平均分，由高到低排列；
   - 若专业计划缺失，依次用录取人数、26 计划、1 作为兜底权重；
   - 若专业分缺失，则用专业组 25 投档线作为兜底，不覆盖原始数据。

数据口径仍保持：所有填充数据以专家版 Excel 行级权威数据为准。
