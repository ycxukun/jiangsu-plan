# 江苏省招生计划变化知识库｜仅标刺客专业｜新增专业组筛选版

## 本版变化

1. 修正上一版过度染色问题。
2. 不再按专业名称全局扫色。
3. 不再把专业组标题合并到每个专业行判断。
4. 只在“有好有坏”的专业组里，把偏离主体的刺客专业标出来。
5. 正常主体专业，如计算机、人工智能、电子信息、电气、自动化等，不再被整组染色。
6. 保留省份/层次多选静态修复、只看新增专业组、特殊培养合并筛选。

## 上传方式

解压后，将所有文件上传到 GitHub 仓库根目录并覆盖旧文件：

- index.html
- data-db.js
- data-major-details.js
- app.js
- .nojekyll
- README.md

不要上传 zip 本身。
