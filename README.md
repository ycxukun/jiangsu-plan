# 江苏省招生计划变化知识库｜V15.3 双入口版

## 两个入口

### 公开展示页

`index.html`

线上地址：

`https://ycxukun.github.io/jiangsu-plan/`

公开只读，不显示备注编辑入口。

### 管理员编辑页

`admin.html`

线上地址：

`https://ycxukun.github.io/jiangsu-plan/admin.html`

打开后左下角直接显示“管理员备注”。

## 管理员功能

1. 新增当前学校备注。
2. 右键专业组/专业新增备注。
3. 连接 GitHub。
4. 同步线上 annotations.json。
5. 固定备注面板。
6. 详情页显示备注。
